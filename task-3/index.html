UX Case Study: Flipkart

Analysis of the user journey, strengths, weaknesses, and proposed UI/UX improvements.

1. Introduction

Flipkart is one of India's largest e-commerce platforms. This report evaluates the shopping experience from landing on the homepage to completing a purchase.

2. User Journey

  1. Open website/app
  2. Search for a product
  3. Apply filters & sort options
  4. View product details
  5. Add to cart
  6. Checkout & payment
  7. Order confirmation & tracking

3. Strengths

  • Fast product search
  • Large catalog
  • Multiple payment methods
  • Clear order tracking
  • Frequent offers

4. Weaknesses

  • Promotional banners can feel crowded
  • Filter panels may overwhelm new users
  • Product comparison is not always prominent
  • Review quality varies
